To start using the Firebase Cloud Messaging (FCM) plugin within your project, import it at the top of your project files:

```dart
import 'package:firebase_messaging/firebase_messaging.dart';
```

Before using Firebase Cloud Messaging, you must first have ensured you have [initialized FlutterFire](overview#initializing-flutterfire).
